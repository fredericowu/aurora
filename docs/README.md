# Aurora Search API Documentation

This directory contains the API documentation hosted on GitHub Pages.

## Setup GitHub Pages (One-time)

To enable GitHub Pages for this repository:

1. Go to: https://github.com/fredericowu/aurora/settings/pages
2. Under "Build and deployment":
   - **Source**: Select "GitHub Actions"
3. Click "Save"

That's it! The documentation will be automatically deployed on every push to `main` that changes the `docs/` folder.

## Accessing the Documentation

Once enabled, the documentation will be available at:
- **https://fredericowu.github.io/aurora/**

## Updating the Documentation

The OpenAPI spec is automatically generated by FastAPI. To update it:

```bash
# Download latest OpenAPI spec from deployed API
curl https://your-api.execute-api.region.amazonaws.com/openapi.json > docs/openapi.json

# Commit and push
git add docs/openapi.json
git commit -m "docs: update OpenAPI spec"
git push
```

The GitHub Actions workflow will automatically deploy the changes.
